<html></html>
웹 브라우저에서 HTML 문서임을 명시하기 위한 태그
<!DOCTYPE ...>
브라우저가 올바르게 웹 페이지를 표시할 수 있도록 해당 문서에 사용한 HTML의 종류와 버전을 표시한다. HTML버전 서로 다른 형태의 <!DOCTYPE>을 사용한다.
<head></head>
웹 브라우저를 통해 사용자에게 보이지는 않지만 문서의 각종 정보와 문서 자체에 대한 설명을 담는 부분이다. <title>,<link>,<style>,<script> 등의 태그를 포함.
<meta charset="ecu-kr"> (UTF-8이 한글 사용을 위해 가장 많이 쓰인다.)
문자 인코딩 선언.
웹 브라우저가 한글 문자를 제대로 인식하지 못하여 글자가 깨지는 현상을 방지하기 위해 한글 폰트 지정 메타 태그를 사용할 수 있다.
<link>
별도의 파일(*.css)로 이루어진 CSS 스타일을 HTML 문서에 적용하기 위해서 사용하는 태그이다.
<base>
링크의 기준이 되는 URL,즉 홈페이지의 기본이 되는 URL을 설정하는 태그이다. 이 태그로 기본 URL을 설정하면 향후 body 요소 내에서 상대경로로 표시한 URL은 절대경로 처럼 인식된다. 
<title></title>
웹 브라우저 상단에 HTML 문서(또는 웹 사이트)의 제목을 보여주는 태그
<!-- -->
주석 태그
<br> 
Forced line break. <br/>로 쓰기도 한다. 줄넘김 태그
<p></p>
paragraph. HTML에서 문단을 나누어 독립된 단락을 생성하는 태그. 웹 브라우저에서는 단락을 구분하기 위해서 단락의 전후에 빈 줄이 자동적으로 추가된다.
공백 및 특수 문자 입력
&lt; = <     &gt; = >  &amp; = &  &quot; = "  &nbsp; = 공백  &#16 = &copy  &#165; = &yen
